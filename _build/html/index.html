Python最佳实践指南！¶

原版

https://pythonguidecn.readthedocs.io/zh/latest/

程序员的核心竞争力是学习能力，而学习能力的核心是技术文档，可以说，技术文档是第一生产力。

Python程序员的养成分4个阶段

  • 入门阶段 - 熟悉Python术语，建立起概念，了解最基本的语法 -> 初级Python工程师

  • Python进阶 - 掌握数据结构和高级特性 -> 中级Python工程师

  • 最佳实践 - 前面两个是纵向提升，而最佳实践是横向拓展，需要同时搭建两个东西 -> 高级Python工程师

    • 通用组件库

    • 设计模式总结

  • 查缺补漏 - 仍然是横向拓展，阅读更多的源码，继续拓展见识 -> Python大师
